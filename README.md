# NutriScan

**NutriScan** é uma plataforma interativa de educação nutricional desenvolvida para capacitar consumidores e acadêmicos na interpretação técnica de rótulos e tabelas nutricionais.

O projeto utiliza gamificação para transformar a leitura de componentes químicos e dados numéricos em uma experiência de aprendizado dinâmica e intuitiva.

## Funcionalidades

- **Maratona Detetive:** 8 desafios técnicos divididos entre 4 produtos populares do mercado.
- **Scanner Óptico Simulado:** Efeito visual de laser para foco na análise de dados.
- **Navegação Didática:** Controle manual de avanço para facilitar explicações em sala de aula.
- **Guia de Resultados:** Infográfico final com diretrizes para aplicação do conhecimento no cotidiano.
- **Visual Minimalista:** Interface "Clean & Pastel" focada na usabilidade e clareza.

## Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização moderna com bordas arredondadas e animações fluidas.
- **JavaScript (Vanilla):** Lógica do jogo e manipulação do DOM.
- **Canvas Confetti:** Biblioteca para efeitos de celebração nos acertos.

## Como executar o projeto

1. Faça o download ou clone este repositório.
2. Certifique-se de que a pasta `assets` contém as imagens dos produtos.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
